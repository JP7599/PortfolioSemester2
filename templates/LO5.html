Learning Outcome 5

Personal Leadership

AI Chatbot

I created a chatbot using the ChatGPT API for my portfolio which you can see at the end of the homepage. The chatbot answers questions about my portfolio and work, redirecting conversations back to relevant topics. This project involved taking the initiative to develop a unique tool to showcase my work, seeking feedback, and iterating on the design and functionality based on that feedback. Through this process, I identified my core values of innovation, user engagement, and clear communication, which have influenced my professional development and the direction of my study career.

Flask Portfolio

I took the initiative to code my portfolio using Flask to make the chatbot work, even though we had only learned HTML, CSS, and JavaScript up to that point. This decision demonstrated my willingness to go beyond the curriculum to enhance my skills. I researched and learned Flask independently, implemented it in my portfolio, and sought feedback to refine it. This experience showcased my commitment to personal growth, self-directed learning, and applying new technologies to improve my work.
